# Data Analytics Projects and Case Studies

This repository contains my data analytics projects and case studies focused on SQL analysis, dashboards, and business insights.

## What you will find here
- SQL analysis and KPI reporting
- Power BI and Tableau dashboards
- Data cleaning, validation, and exploratory analysis
- Case studies with clear problem statements and findings

## Skills and tools
SQL | Python | Power BI | Tableau | Excel  
pandas | numpy | matplotlib | scikit-learn (when needed)

## Repository structure
- `SQL/` : SQL queries, schema notes, and KPI logic
- `PowerBI/` : dashboard files, measures, and screenshots
- `Tableau/` : workbooks and screenshots
- `Python/` : notebooks for EDA and analysis
- `Reports/` : summaries of insights and recommendations

## How to use
1. Start with the folders above.
2. Each project folder includes a short `README.md` with context, approach, and results.
3. Screenshots are included for dashboards where available.
